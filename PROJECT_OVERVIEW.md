## 🎯 核心功能

### 1. 按鈕注入
- 在每個分類行自動添加「📁 移動到 ▼」按鈕
- 支援動態注入（MutationObserver）
- 特殊分類禁用檢查

### 2. 智能下拉選單
- 層級感知的目標計算
- 樹狀結構顯示（├─ 縮排）
- 自動排除無效目標
- 智能定位（考慮視口邊界）

### 3. 雙方案移動
- **主方案**: AngularJS $scope 操作（高效）
- **備案方案**: DragEvent 模擬（容錯）
- 自動容錯切換

### 4. 使用者回饋
- 成功訊息：綠色 Toast（2 秒）
- 失敗訊息：紅色 Toast（3 秒）
- 清晰的錯誤說明

### 5. 自動持久化
- 變更自動保存到後台
- 支援頁面重新整理後保持
